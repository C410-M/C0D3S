# ***Downloader_YoutuBIU***

Cançado de enfrentar Sites lentos com vários Spams, para baixar um arquivo do Youtube?

Baixe a versão .PY, ignore os anti-vírus e seja feliz baixando Áudios e Vídeos instântaneamente pelo seu terminal :)

Instruções:

1- Duplo clique no arquivo executável (downloader_YoutuBIU.exe)

2- Pressiona enter pra iniciar

3- Informe a URL do vídeo que deseja baixar e converter em MP3

4- Informe o caminho da pasta, onde deseja salvar o arquivo. Ou apenas aperte Enter para salvar na pasta atual. 

Ex: D:\Users\Downloads

5- Aproveite o som 🎚️😎
